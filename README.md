# Telegram Mini App: Калькулятор депозита

Файлы:
- index.html — страница приложения
- style.css — дизайн
- app.js — логика расчёта
- bot-button-example.js — пример кнопки для Telegram-бота

Что менять:
- В `index.html` можно заменить название «Дом Трейдеров».
- В `app.js` можно поменять формулу режима `martingale`.
- В `.env` бота нужно указать `MINI_APP_URL` — ссылку на загруженное приложение.
